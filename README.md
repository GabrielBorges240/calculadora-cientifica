# Calculadora Científica

Calculadora científica com operações básicas e avançadas, focada em lógica, validação e interface moderna.

Demo:  https://gabrielborges240.github.io/calculadora-cientifica/

## Funcionalidades
- Operações básicas (+, -, ×, ÷)
- Funções científicas (potência, raiz, porcentagem)
- Tratamento de erros matemáticos
- Interface responsiva

## Tecnologias
- HTML5
- CSS3
- JavaScript

## Como executar
Abra o arquivo `index.html` no navegador.
